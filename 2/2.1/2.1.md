### Módulo 1: Gestión de Usuarios

| ID   | Nombre                             | Historia de Usuario                                                                                                                                                     |
|------|------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| RF1  | Registro de usuario                | Como usuario, quiero poder registrarme en la plataforma completando un formulario con mis datos personales, para crear mi cuenta y empezar a usar Singletone.             |
| RF2  | Validación de correo y nickname    | Como usuario, quiero que el sistema me avise si mi correo o nickname ya están registrados, para poder corregirlos y completar mi registro sin inconvenientes.            |
| RF3  | Inicio de sesión                   | Como usuario, quiero poder iniciar sesión usando mi correo o nickname y mi contraseña, para acceder de forma segura a mi cuenta en cualquier momento.                    |
| RF4  | Visualización de tipo de cuenta    | Como usuario, quiero que el sistema reconozca si mi cuenta es gratuita o premium al iniciar sesión, para tener claridad sobre las funciones disponibles.                  |
| RF5  | Activación de cuenta premium       | Como usuario que selecciona el plan premium, quiero ser dirigido automáticamente al sistema de pagos, para poder completar mi suscripción antes de usar la plataforma.    |
| RF6  | Seguridad en mis datos             | Como usuario, quiero que mis datos personales y mi contraseña estén protegidos, para sentir confianza en el uso de la plataforma y en el manejo de mi información.        |

### Módulo 4: Valoración Músical

| ID  | Nombre                                | Historia de Usuario |
|-----|----------------------------------------|----------------------|
| 4.1 | Límite de valoraciones semanales      | Como usuario gratuito, quiero valorar hasta 7 álbumes por semana, para poder hacer máximo uso de mi plan gratuito. |
| 4.2 | Valoración de canciones individuales de un álbum | Como usuario, quiero calificar canción por canción dentro de un álbum, para que el sistema calcule una valoración más precisa. |
| 4.3 | Botón de guardar valoración y actualización de estado | Como usuario, quiero guardar mis valoraciones y que el sistema marque el álbum como “valorado” si completé todo, para tener un seguimiento de mi actividad. |
| 4.4 | Eliminar artista de la biblioteca del usuario | Como usuario, quiero eliminar artistas específicos de mi biblioteca y junto con ello sus álbumes respectivos, para así remover artistas que ya no me interesan o me equivoqué al agregarlos. |
| 4.5 | Revaloración de álbumes sin límite    | Como usuario, quiero poder editar las valoraciones de canciones cuando cambio de opinión, pero sin alterar el año de escucha. |

###  Módulo 5: Gestión de Recomendaciones


| ID  | Nombre                                | Historia de Usuario |
|-----|----------------------------------------|----------------------|
| 5.1 | Generación de recomendaciones personalizadas | Como usuario premium, quiero recibir recomendaciones personalizadas basadas en mi historial musical, para descubrir nuevos artistas y álbumes que se ajusten a mis gustos. |
| 5.2 | Sección de recomendaciones en la página de inicio | Como usuario premium, quiero ver sugerencias destacadas en mi inicio, para descubrir contenido relevante sin tener que buscarlo manualmente. |
| 5.3 | Re-generación de recomendaciones personalizadas | Como usuario premium, quiero poder solicitar la generación de nuevas recomendaciones para mí, en caso de que las ya generadas no llamen mi interés. |
| 5.4 | Integración de sistema inteligente     | Como administrador del sistema, quiero que las recomendaciones sean generadas por un modelo inteligente de ML o una API de LLM, para asegurar que sean relevantes y personalizadas según el comportamiento del usuario. |
| 5.5 | Volver a recomendar                    | Como usuario premium, quiero poder actualizar mis recomendaciones si no me convencen las actuales, para tener más opciones musicales alineadas a mis intereses. |

###  Módulo 6: Gestión de Planes y Monetización

| ID  | Nombre                                | Historia de Usuario |
|-----|----------------------------------------|----------------------|
| 6.1 | Visualización de banner de promoción para usuarios gratuitos | Como usuario gratuito, quiero ver un banner con información sobre el plan premium, para conocer sus beneficios y considerar la suscripción. |
| 6.2 | Simulación de pasarela de pago        | Como usuario, quiero realizar el pago de mi suscripción de forma sencilla mediante una interfaz simulada, para comenzar a usar las funciones premium. |
| 6.3 | Selección de meses de suscripción con descuentos progresivos | Como usuario, quiero poder seleccionar cuántos meses deseo pagar por adelantado y recibir un descuento si elijo un periodo más largo, para ahorrar dinero. |
| 6.4 | Activación de funciones premium tras pago exitoso | Como usuario, quiero que mi cuenta se actualice inmediatamente al estado premium después de realizar el pago, para comenzar a disfrutar los beneficios del plan premium. |
| 6.5 | Visualización del estado premium en el perfil | Como usuario premium, quiero que mi perfil refleje mi tipo de cuenta y muestre cuándo termina mi suscripción, para tener control sobre su renovación. |
